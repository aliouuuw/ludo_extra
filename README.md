This is a [vinext](https://github.com/mjackson/vinext) project migrated from Next.js. It uses the Vite-based implementation of the Next.js App Router API.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [Geist](https://vercel.com/font) font family loaded from Google Fonts.

## Deploy

This project can be deployed to Cloudflare Workers using `vinext deploy`.
